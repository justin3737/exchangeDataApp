Exchange Data App
===
### Description
- This is a test project using react and redux data flow.
- Run test rest api to get json data and display in the project.
- Use app's ui to exchange data.

### Usage
Step.0 安裝套件
```
npm install
```

Step.1 起服務server

```
npm run server
```

Step.2 開啟新的Terminal (command + T)接著起專案

```
npm run start
```

Step.3 在瀏覽器開啟

```
http://localhost:3030
```
